# 💳 PocketPay - Fintech Web App

A simple and responsive digital wallet web application inspired by bKash/Nagad.

## 🚀 Features
- Add Money
- Cashout
- Send Money
- Pay Bills
- Get Bonus
- Transaction History

## 🛠 Tech Stack
- HTML5
- Tailwind CSS + DaisyUI
- JavaScript

## 🎯 Highlights
- Clean UI/UX
- Mobile responsive
- Real-time balance update
- Dynamic transaction system
